# Geralt
Geralt is an upcoming cryptographic library for [.NET](https://dotnet.microsoft.com/) based on [libsodium](https://doc.libsodium.org/). It is a cleaned up fork of [libsodium-core](https://github.com/tabrath/libsodium-core).
