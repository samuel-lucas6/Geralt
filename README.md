# Geralt
Geralt is a cryptographic library for [.NET](https://dotnet.microsoft.com/) based on [libsodium](https://doc.libsodium.org/). Geralt is a cleaned up fork of [libsodium-core](https://github.com/tabrath/libsodium-core).
